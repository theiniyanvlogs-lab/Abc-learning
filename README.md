# ABC Learning App

Fun ABC learning app for kids with:
- A-Z letters
- Voice speech
- Grok AI integration
- Kid photo upload
- Camera support on mobile
- Mobile + desktop support
- Vercel ready

## Run locally

```bash
npm install
npm run dev
```

## Add environment variable

Create `.env.local`

```env
GROK_API_KEY=your_grok_api_key_here
```

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Add `GROK_API_KEY` in Vercel settings
4. Deploy
